Instalación de Git

Abre el terminal y ejecuta el comando correspondiente para instalar Git: brew install git (para OS X), git --version (para Windows) o sudo dnf install git o sudo yum install git (para Linux)

que diferencia tiene git y github

En resumen, la principal diferencia entre git y GitHub es que Git es el sistema de control de versiones en sí mismo, mientras que GitHub es un servicio en línea que utiliza Git para almacenar repositorios
